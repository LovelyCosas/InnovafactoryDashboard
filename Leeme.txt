# InnovaFactory Dashboard

## 📋 Descripción
InnovaFactory Dashboard es una aplicación web para monitorizar en tiempo real sensores industriales. Permite visualizar datos de temperatura, vibración y humedad en diferentes zonas de la planta, con alertas en tiempo real y seguimiento histórico.

## 🚀 Características
- **Monitorización en Tiempo Real**: Visualización de hasta 10 sensores por zona
- **Sistema de Alertas**: Notificaciones inmediatas cuando los sensores superan umbrales críticos
- **Histórico de Lecturas**: Registro de las últimas mediciones
- **Gráfico de Evolución**: Seguimiento de temperatura en la última hora
- **Filtrado por Zonas**: Selección de diferentes áreas de la planta
- **Panel de Usuario**: Acceso a funcionalidades de mantenimiento

## 🛠️ Tecnologías Utilizadas
- **Frontend**: React.js
- **Backend**: Node.js + Express
- **Gráficos**: Chart.js
- **Estilizado**: CSS3 con diseño responsive

## ⚙️ Instalación

### Prerequisitos
- Node.js (v14 o superior)
- npm (v6 o superior)

### Pasos de Instalación

1. **Clonar el repositorio**
   ```bash
   git clone [url-del-repositorio]
   ```

2. **Instalar dependencias del backend**
   ```bash
   cd backend
   npm install
   ```

3. **Instalar dependencias del frontend**
   ```bash
   cd ../frontend
   npm install
   ```

4. **Configurar variables de entorno**
   - Crear archivo `.env` en el directorio backend
   ```env
   PORT=3000
   ```

5. **Iniciar la aplicación**
   
   Backend:
   ```bash
   cd backend
   npm run dev
   ```

   Frontend:
   ```bash
   cd frontend
   npm start
   ```

## 🖥️ Uso

1. **Acceder a la aplicación**
   - Abrir navegador en `http://localhost:3000`

2. **Monitorización de Sensores**
   - Los sensores se muestran en la parte superior
   - Se actualizan automáticamente cada 5 segundos
   - Las alertas aparecen en rojo

3. **Filtrado por Zonas**
   - Usar el selector de zona para filtrar sensores
   - Zonas disponibles:
     - Almacén A
     - Zona B
     - Zona C
     - Motor A
     - Motor B
     - Horno A
     - Zona Técnica

4. **Panel de Usuario**
   - Acceder al menú de usuario desde el icono superior derecho
   - Funcionalidades disponibles:
     - Mi Perfil
     - Mis Tareas
     - Órdenes de Trabajo
     - Calendario
     - Informes
     - Configuración

## 📊 Datos Técnicos

### Umbrales de Alerta
- Temperatura: > 80°C
- Vibración: > 0.8 m/s²
- Humedad: > 85%

### Intervalos de Actualización
- Sensores: 5 segundos
- Histórico: 30 segundos
- Gráfico: Tiempo real
